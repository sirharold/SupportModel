Esa larga base de documentos, no siempre es leída completamente. 

El propblema que se quiere resolver es poder brindar una serie de links a documentos existentes, basados en la pregunta que se realiza. 

Este proyecto es la tesis de grado para obtener un magister en data science.

El tono del proyecto debe ser profesional - académico
